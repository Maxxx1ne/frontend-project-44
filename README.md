# Игра в терминале
Для привествия необходимо прописать команду:

node bin/brain-games.js



Далеее нужно выбрать в какую игру вы хотите поиграть:

'calc' - калькулятор

'even' - четные/нечетные числа

'gcd' - наибольший общий делитель

'prime' - простое/непростое число

'progression' - прогрессия чисел



Теперь когда вы выбрали во что хотите сыграть пропишите команду в терминале:

node bin/brain-'название игры которую вы выбрали'.js


Например:


Для того чтобы запустить игру 'калькулятор' нужна команда:

node bin/brain-calc.js



Выбирайте игру и играйте с удовольствием!!!
