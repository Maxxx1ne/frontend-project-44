# Игры в терминале

Добро пожаловать в коллекцию игр для терминала! Здесь вы можете выбрать и сыграть в различные увлекательные игры. 

## Запуск игры

Для начала работы с играми выполните следующую команду в терминале:

bash
node bin/brain-games.js

После этого вам будет предложено выбрать игру, в которую вы хотите поиграть. Доступные игры:

- **calc** - калькулятор
- **even** - четные/нечетные числа
- **gcd** - наибольший общий делитель
- **prime** - простое/непростое число
- **progression** - прогрессия чисел

### Как запустить игру

После выбора игры введите соответствующую команду в терминале:

bash
node bin/brain-'название игры'.js

Например, чтобы запустить игру "калькулятор", используйте команду:

bash
node bin/brain-calc.js

Выбирайте игру и наслаждайтесь игрой!

## Примеры запуска игр

- Для игры "четные/нечетные числа":

bash
node bin/brain-even.js

Надеемся, вам понравится играть в наши игры! Удачи!
