# Игры в терминале

Добро пожаловать в коллекцию игр для терминала! Здесь вы можете выбрать и сыграть в различные увлекательные игры. 

## Запуск приветствия

Для начала работы с играми выполните следующую команду в терминале:

bash

npm install readline-sync

## Старт игры

Вам нужно вписать в терминал команду вызова приветствия:

bash

make brain-games

### Выбор игры

Теперь вам нужно выбрать игру в которую вы хотите поиграть:

- **calc** - калькулятор
- **even** - четные/нечетные числа
- **gcd** - наибольший общий делитель
- **prime** - простое/непростое число
- **progression** - прогрессия чисел

### Как запустить игру

После выбора игры введите соответствующую команду в терминале:

bash

make brain-'название игры'

#### Например

Чтобы запустить игру "калькулятор", используйте команду:

bash

make brain-calc

#### Выбирайте игру и наслаждайтесь ею!
